Python Project


Project Description:
The Grade Evaluation System is a Python-based application designed to evaluate the performance of students based on their marks in a given set of exams. The system classifies students' performance into three categories: "Pass," "Fail," and "Defer." It operates based on a set of rules that consider multiples of 20, with a maximum total mark of 120.

Java Project


The Food Management System is a Java and JavaFX-based application designed for a burger shop. It aims to streamline and automate the ordering and payment process. The system is primarily focused on managing orders from three cashiers, each with different capabilities: Cashier 1, Cashier 2, and Cashier 3. Each cashier can serve a maximum of 2, 3, and 5 customers simultaneously, respectively. Key Features: Cashier Management: Cashier Assignment: Customers are assigned to available cashiers based on their capacity. Capacity Limits: Cashiers have a limit on the number of customers they can serve simultaneously. Ordering System: Menu Selection: Customers can choose from a variety of burger options. Quantity Limit: Each customer can order a maximum of 10 burgers. Pricing and Payment: Price Calculation: The system calculates the total price of the order. Payment Handling: Cashiers can record and manage payments from customers. Total Income Tracking: Income Calculation: The system keeps a record of the total income generated from all orders. Order Status: Order Queue: Orders are processed in a first-come, first-served manner. Status Updates: Customers can view the status of their orders, such as "Order Received," "Preparing," "Ready for Pickup." User-Friendly Interface: JavaFX GUI: The system features an intuitive graphical user interface for easy interaction
